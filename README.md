## Getting Started

To get a local copy up and running follow these setup steps.

**Setup**

- Open terminal and clone the project: `git clone git@github.com:Mo7amedSMahdi/react-todo.git`
- cd into the directory `react-todo`

**Prerequisites**

- You need to have `node.js` installed in your system

**Install**

- On your terminal run :

```sh
  npm install
```

- then build :

```sh
npm run build
```

- In case you run into any errors please raise an [issue](https://github.com/Mo7amedSMahdi/art-institute-chicago-api/issues).

**Usage**

- To run the application:

```sh
npm start

```
